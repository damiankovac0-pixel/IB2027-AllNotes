# A1.1 Computer Hardware And Operation

This topic explains how the physical parts of a computer work together: the [[A1.1.1 CPU components|CPU]], [[A1.1.2 GPU role|GPU]], memory, storage, instruction execution, compression, and cloud services.

The big idea is that a computer system works by moving instructions and data between processing units, memory, storage, and external services.

## Core Distinction

Primary memory is directly involved in active processing. Secondary storage keeps data long-term. The CPU executes instructions. The GPU accelerates highly parallel tasks. Cloud services move some computing resources away from local hardware and onto remote infrastructure.
