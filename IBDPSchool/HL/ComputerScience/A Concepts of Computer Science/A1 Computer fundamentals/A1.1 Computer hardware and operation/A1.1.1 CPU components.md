# A1.1.1 CPU Components

The CPU is the part of the computer that executes instructions. It coordinates the movement of data, performs calculations, and controls the rest of the system.

## Main CPU Components

### Control Unit

The control unit, or CU, directs the CPU. It does not usually perform calculations itself. Instead, it controls timing and tells other components what to do.

It coordinates:

- fetching instructions from memory
- decoding instructions
- controlling data movement
- sending read/write signals
- managing the fetch-decode-execute cycle

### Arithmetic Logic Unit

The arithmetic logic unit, or ALU, performs arithmetic and logical operations.

Arithmetic examples:

- addition
- subtraction
- multiplication
- division

Logic examples:

- comparisons such as greater than or equal to
- Boolean operations such as AND, OR, and NOT

### Registers

Registers are very small, very fast storage locations inside the CPU.

Important registers:

| Register | Role |
| --- | --- |
| Program counter, PC | stores the address of the next instruction |
| Instruction register, IR | stores the current instruction being executed |
| Memory address register, MAR | stores the memory address being accessed |
| Memory data register, MDR | stores data being transferred to or from memory |
| Accumulator, AC | stores intermediate calculation results |

## Buses

Buses are pathways that transfer information between CPU, memory, and other components.

### Data Bus

The data bus carries actual data. It is usually bidirectional because data can move to and from the CPU.

### Address Bus

The address bus carries memory addresses. It tells memory where data should be read from or written to.

### Control Bus

The control bus carries control signals such as read, write, timing, interrupt, and acknowledgement signals.

## Cores

A core is a processing unit within a CPU.

### Single-Core Processor

A single-core CPU has one processing unit. It can still appear to run multiple programs by rapidly switching between tasks, but it only executes one stream of instructions at a time.

### Multi-Core Processor

A multi-core CPU has two or more cores. It can run multiple instructions or tasks more genuinely in parallel, if the software is designed to use multiple cores.

Examples:

- dual-core: 2 cores
- quad-core: 4 cores
- hexa-core: 6 cores
- octa-core: 8 cores

## How The Components Interact

A simple read from memory works like this:

1. The CPU places the required memory address in the MAR.
2. The address bus sends the address to memory.
3. The control bus sends a read signal.
4. Memory returns the data through the data bus.
5. The data is stored in the MDR.
6. The CPU can then use the data, often through the ALU or registers.

## Exam Pattern

To describe CPU interaction, mention the register involved, the bus used, and the direction of data or control signals.
