# A1.1.2 GPU Role

A GPU, or graphics processing unit, is a specialized processor designed for tasks that can be split into many similar calculations running at the same time.

GPUs were originally built for graphics rendering, but they are now also important in [[A4.1.2 Hardware for machine learning#GPU|machine learning]], scientific computing, video processing, and cryptocurrency mining.

## Why GPUs Are Good At Graphics

Images and videos contain many pixels. The same kind of calculation often has to be applied to thousands or millions of pixels.

A GPU can process many pixels simultaneously because it has many small cores optimized for parallel work.

Graphics tasks include:

- rendering frames
- processing textures
- applying shaders
- calculating lighting
- handling vertex and pixel data
- video editing and playback acceleration

## Parallel Processing

Parallel processing means performing multiple calculations at the same time.

This is the GPU's main advantage. Instead of one powerful core doing one complex task, a GPU uses many smaller cores to perform similar operations across large amounts of data.

## Beyond Graphics

GPUs are useful when the workload can be split into many independent parts.

Examples:

- training neural networks
- matrix and vector multiplication
- scientific simulations
- video encoding
- image filtering
- some cryptographic or blockchain-related tasks

## GPU In Machine Learning

Machine learning, especially [[A4.1.1 Types of machine learning#Deep Learning|deep learning]], uses many matrix and vector operations. GPUs speed this up because the same mathematical operations can be applied to many data values in parallel.

This is why GPUs are a major part of [[A4.1.2 Hardware for machine learning|machine learning hardware]].

## CPU And GPU In Games

In a video game:

- the CPU handles game logic, rules, physics, AI behaviour, and user input
- the GPU renders the visuals, frames, textures, lighting, and effects

The game feels smooth when CPU and GPU work together effectively.

## Exam Pattern

A GPU is suitable for graphics and machine learning because it contains many cores optimized for parallel processing, allowing it to perform the same operation on many data points simultaneously.
