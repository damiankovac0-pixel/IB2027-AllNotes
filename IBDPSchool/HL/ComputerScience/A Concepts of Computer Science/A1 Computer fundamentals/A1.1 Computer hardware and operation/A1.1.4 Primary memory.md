# A1.1.4 Primary Memory

Primary memory is memory directly used by the CPU during active processing.

The main types are:

- RAM
- ROM
- cache
- registers, covered in [[A1.1.1 CPU components#Registers|CPU components]]

## RAM

RAM stands for random access memory.

RAM stores the operating system, programs, instructions, and data that are currently in use.

Important properties:

- volatile: contents are lost when power is off
- fast compared with secondary storage
- temporary workspace for active processes

Example: when an app is opened, it loads from secondary storage into RAM so the CPU can access it quickly.

## ROM

ROM stands for read-only memory.

ROM stores essential startup instructions, such as firmware or BIOS.

Important properties:

- non-volatile: contents remain when power is off
- usually not changed during normal use
- needed to start the computer and load the operating system

## Cache Memory

Cache is very fast memory close to or inside the CPU.

It stores frequently used instructions and data so the CPU can access them faster than going to RAM.

Typical cache levels:

- L1: smallest and fastest, closest to the core
- L2: larger but slower than L1
- L3: larger again, often shared between cores, slower than L1/L2 but still faster than RAM

## Cache Hit And Cache Miss

A cache hit occurs when the CPU finds the required data in cache.

A cache miss occurs when the CPU does not find it in cache and must fetch it from slower memory.

Cache hits improve performance. Cache misses slow processing down.

## Cache Techniques

Systems try to improve cache performance using:

- prefetching: predicting data needed soon and loading it early
- cache replacement policies: choosing what to remove when cache is full
- memory allocation strategies: arranging data to improve cache hits

## RAM vs ROM

| Feature | RAM | ROM |
| --- | --- | --- |
| Volatile | yes | no |
| Main use | active programs and data | startup instructions |
| Changed during normal use | yes | usually no |
| Speed | fast | less about active speed, more about persistence |

## Exam Pattern

RAM is volatile working memory for active programs. ROM is non-volatile memory for startup instructions. Cache is very fast memory used to reduce CPU waiting time by storing frequently accessed data and instructions.
