# A1.1.7 Secondary Memory Storage

Secondary memory is non-volatile storage used to keep data long-term.

Unlike RAM, it retains data when the computer is powered off.

## Internal And External Storage

Internal storage is built into the device.

External storage connects from outside the device.

Examples:

- internal HDD or SSD
- external USB drive
- memory card
- optical disc
- network attached storage

## HDD

An HDD, or hard disk drive, stores data magnetically on spinning disks.

Strengths:

- high capacity
- low cost per GB
- good for backups and large media files

Weaknesses:

- slower than SSDs
- moving parts can be damaged
- less suitable for portable devices

## SSD

An SSD, or solid state drive, uses flash memory and has no moving parts.

Strengths:

- fast read/write speeds
- durable
- quiet
- good for operating systems, software, and games

Weaknesses:

- more expensive per GB than HDDs
- limited write cycles, though modern SSDs manage this well

## eMMC

Embedded multimedia card storage is flash storage often used in low-cost devices.

It is cheaper and simpler than a full SSD, but usually slower.

Common in entry-level phones, tablets, and low-cost laptops.

## Optical Storage

Optical storage includes CDs, DVDs, and Blu-ray discs.

Data is read using a laser that detects pits and lands on the disc surface.

Strengths:

- useful for distribution or archival storage
- removable
- low cost

Weaknesses:

- slower than SSDs
- can be scratched
- lower capacity than modern drives
- optical drives are less common now

## Flash Drives And Memory Cards

Flash drives and memory cards use flash memory.

Strengths:

- portable
- no moving parts
- convenient for cameras, phones, and file transfer

Weaknesses:

- small and easy to lose
- variable reliability
- usually slower than high-quality SSDs

## NAS

NAS stands for network attached storage.

It is storage connected to a network so multiple devices can access it.

Uses:

- shared files
- backups
- media storage
- business storage

Advantages:

- centralized storage
- shared access
- expandable
- can use RAID for protection against drive failure

Disadvantages:

- more expensive than a simple external drive
- requires setup and network access
- performance depends on the network

## Exam Pattern

Compare storage by volatility, speed, capacity, cost, durability, portability, and use case. HDDs are cheap and high-capacity; SSDs are fast and durable; optical media is removable; NAS is shared network storage.
