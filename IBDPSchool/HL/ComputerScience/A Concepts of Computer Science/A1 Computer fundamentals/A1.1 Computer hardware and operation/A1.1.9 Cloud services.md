# A1.1.9 Cloud Services

Cloud computing provides computing resources over the internet instead of relying only on local hardware.

Resources can include:

- storage
- software
- servers
- databases
- networking
- development platforms
- computing power

## Why Use Cloud Computing?

Organizations use cloud services because they can rent resources instead of buying and maintaining all hardware themselves.

Benefits:

- scalability
- lower upfront cost
- pay-as-you-use pricing
- remote access
- easier collaboration
- managed updates and maintenance

Limitations:

- dependence on internet access
- ongoing subscription costs
- privacy and security concerns
- reliance on the provider
- possible vendor lock-in

## SaaS

SaaS means software as a service.

The provider hosts the software, and users access it through a browser or app.

Examples:

- Google Docs
- Microsoft 365 web apps
- Salesforce
- MoodleCloud

Best for users who want ready-to-use software without managing installation, servers, or updates.

## PaaS

PaaS means platform as a service.

The provider supplies a platform for developers to build, test, and deploy applications.

The developer manages the application code, while the provider manages much of the underlying infrastructure.

Examples:

- app hosting platforms
- database platforms
- development environments

Best for developers who want to build applications without managing servers directly.

## IaaS

IaaS means infrastructure as a service.

The provider supplies virtualized infrastructure such as virtual machines, storage, and networking.

The customer manages operating systems, software, and applications.

Examples:

- cloud virtual machines
- cloud storage volumes
- virtual networks

Best for organizations that want flexible infrastructure without buying physical servers.

## SaaS vs PaaS vs IaaS

| Service | Provider manages | User manages | Example use |
| --- | --- | --- | --- |
| SaaS | almost everything | using the software | email, documents, CRM |
| PaaS | platform and infrastructure | application code | building web apps |
| IaaS | virtual hardware | OS, software, apps | renting virtual servers |

## Link To Machine Learning

Cloud computing is important in [[A4.1.2 Hardware for machine learning#Cloud-Based Platforms|machine learning hardware]] because large models may need scalable GPUs, TPUs, RAM, and storage that are easier to rent than buy.

## Exam Pattern

When comparing cloud services, explain what is managed by the provider and what is managed by the user. SaaS gives complete software, PaaS gives a development platform, and IaaS gives virtual infrastructure.
