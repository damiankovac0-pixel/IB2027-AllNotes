# A1.2 Data Representation And Computer Logic

This topic explains how computers represent data using binary and how binary logic is used to build digital circuits.

The big idea: computers store and process everything as patterns of `0` and `1`. These patterns can represent numbers, text, images, audio, video, and logic states.

## Study Order

1. Learn binary, decimal, hexadecimal, bits, bytes, and nibbles.
2. Learn how different data types are stored using binary.
3. Learn each logic gate and its truth table.
4. Practise constructing truth tables from descriptions and diagrams.
5. Practise drawing logic diagrams from Boolean expressions.
