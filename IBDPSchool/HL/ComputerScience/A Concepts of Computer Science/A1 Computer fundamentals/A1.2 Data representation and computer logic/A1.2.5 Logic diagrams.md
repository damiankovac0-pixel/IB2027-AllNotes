# A1.2.5 Logic Diagrams

Logic diagrams show how logic gates are connected to produce an output.

They can be built from Boolean expressions or analysed to create truth tables.

## From Boolean Expression To Diagram

Read the expression in stages.

Example:

```text
Q = (A AND B) OR (A AND NOT B)
```

Steps:

1. Create inputs `A` and `B`.
2. Build `A AND B` with an AND gate.
3. Build `NOT B` with a NOT gate.
4. Build `A AND NOT B` with another AND gate.
5. Feed both AND outputs into an OR gate.
6. Label the final output `Q`.

## From Diagram To Expression

Read from left to right.

Steps:

1. Label inputs.
2. Label intermediate gate outputs.
3. Write the expression for each intermediate output.
4. Combine them into the final expression.

## Simplifying Logic

Simplifying logic means creating an equivalent expression using fewer gates.

Benefits:

- fewer components
- lower cost
- less power use
- faster circuits
- easier debugging
- higher reliability

## Boolean Algebra Rules

Useful simple rules:

| Rule | Meaning |
| --- | --- |
| `A AND A = A` | repeated AND can simplify |
| `A OR A = A` | repeated OR can simplify |
| `NOT NOT A = A` | double negation cancels |
| `A AND 1 = A` | AND with true keeps A |
| `A OR 0 = A` | OR with false keeps A |
| `A AND 0 = 0` | AND with false is false |
| `A OR 1 = 1` | OR with true is true |

## Combining Gates

Complex circuits are built by combining simple gates.

Examples:

- adders use combinations of gates to add binary digits
- memory circuits use gates to store bits
- safety systems use gates to enforce multiple conditions
- timing circuits use gates to control when signals occur

## Full Adder Idea

A full adder adds two binary digits and a carry-in from a previous addition.

It produces:

- sum
- carry-out

You do not need to decode highly complex adder diagrams in detail, but understand that arithmetic circuits are built from logic gates.

## Exam Pattern

When drawing a logic diagram, preserve the order of operations and use brackets as stages. When analysing a diagram, add intermediate labels and work left to right.
