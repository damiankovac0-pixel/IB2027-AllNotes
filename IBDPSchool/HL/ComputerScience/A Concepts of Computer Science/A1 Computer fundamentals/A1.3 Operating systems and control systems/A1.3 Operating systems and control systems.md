# A1.3 Operating Systems And Control Systems

This topic explains how operating systems manage hardware and software, and how control systems use inputs, processing, outputs, and feedback to affect the real world.

The big idea: an operating system hides hardware complexity from users and applications, while a control system uses sensor data to make decisions and trigger physical actions.
