# A1.3.2 Functions Of An Operating System

An operating system performs many functions to keep the computer stable, usable, secure, and efficient.

## Process Management

A process is a program in execution.

The OS tracks each process using information such as:

- process ID
- current state
- priority
- register values
- memory usage
- files and resources being used

Common process states:

| State | Meaning |
| --- | --- |
| New | process is being created |
| Ready | waiting for CPU time |
| Running | currently executing on CPU |
| Blocked or waiting | waiting for an event such as input or disk access |
| Terminated | finished and being cleaned up |

## Memory Management

The OS manages RAM so processes have memory to run safely.

Memory management includes:

- allocating memory to processes
- freeing memory when no longer needed
- preventing one process from accessing another process's memory without permission
- using virtual memory when physical RAM is insufficient

## Virtual Memory

Virtual memory lets the computer behave as if it has more memory than the physical RAM available.

The OS temporarily moves data between RAM and secondary storage.

Benefits:

- allows larger programs to run
- supports multitasking
- gives each process the illusion of a large continuous address space

Limitation: using storage as memory is slower than using RAM.

## File Management

The OS organizes files and directories.

File management includes:

- creating files and folders
- reading and writing files
- deleting, renaming, moving, and copying files
- enforcing naming rules
- managing file permissions
- tracking where files are stored on disk

## File System

A file system defines how data is stored and retrieved on storage devices.

Examples include NTFS, APFS, ext4, FAT32, and exFAT.

The file system manages:

- directory structure
- file metadata
- free space
- permissions
- file location on storage

## Security

OS security protects confidentiality, integrity, and availability.

Security functions include:

- user authentication
- access control
- permissions
- encryption support
- firewall integration
- antivirus integration
- patch and update management
- logging and auditing

## Accounting And Logging

The OS can record resource usage and system activity.

Uses:

- auditing
- detecting suspicious activity
- billing or chargeback in shared systems
- resource planning

## User Interface

The OS provides interfaces such as:

- graphical user interface, or GUI
- command-line interface, or CLI

GUI elements include windows, icons, menus, pointers, progress bars, notifications, and tooltips.

## Virtualization

Virtualization allows multiple virtual machines to run on one physical computer.

A hypervisor creates and manages virtual machines.

Benefits:

- better use of hardware resources
- isolation between virtual machines
- easier testing and deployment
- reduced infrastructure costs

## Networking

The OS manages network communication through network hardware, drivers, and protocol stacks.

It helps with:

- sending and receiving packets
- managing network adapters
- applying network settings
- supporting protocols
- sharing files or services over a network

## Exam Pattern

For OS function questions, name the function and describe what the OS actually manages. Good answers use examples such as process states, virtual memory, file systems, permissions, and networking.
