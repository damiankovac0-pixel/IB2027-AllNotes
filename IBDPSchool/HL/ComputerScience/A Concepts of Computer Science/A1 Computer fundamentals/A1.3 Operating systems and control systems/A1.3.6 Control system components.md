# A1.3.6 Control System Components

A control system monitors conditions, processes input, and produces output that affects a system.

Most control systems follow an input-process-output-feedback pattern.

```text
input -> process -> output -> feedback -> adjustment
```

## Sensors

Sensors detect and measure physical quantities.

Examples:

- temperature sensor
- humidity sensor
- pressure sensor
- light sensor
- proximity sensor
- gas sensor
- touch sensor

Sensors provide input data to the controller.

## Controller

The controller processes input data and decides what action should happen.

It uses a control algorithm to compare current conditions with desired conditions.

Example: a thermostat controller compares room temperature with the target temperature.

## Control Algorithm

A control algorithm is the set of rules or calculations the controller uses to decide output.

Examples:

- simple threshold rule: if temperature is below target, turn heating on
- proportional control: adjust output based on size of error
- PID control: uses present, past, and predicted future error for more precise adjustment

## Actuators

Actuators carry out physical actions based on controller commands.

Examples:

- motor
- heater
- pump
- valve
- brake
- steering mechanism
- door mechanism

## Transducers

A transducer converts one form of energy or signal into another.

Sensors and actuators can both involve transduction.

Example: a microphone converts sound waves into electrical signals.

## Feedback

Feedback is information about the system's output that is returned to the controller.

It lets the controller adjust future actions.

Example: a thermostat checks the current temperature after heating has been turned on.

## Open-Loop Control

An open-loop system does not use feedback to adjust its output.

Example: a basic toaster heats for a fixed time regardless of how toasted the bread is.

## Closed-Loop Control

A closed-loop system uses feedback to adjust output.

Example: cruise control measures current speed and adjusts the throttle to maintain the set speed.

Closed-loop systems are usually better when accuracy and changing conditions matter.

## Exam Pattern

When describing a control system, identify the input sensors, controller, control algorithm, output actuators, and feedback loop. Do not just list components; explain how they interact.
