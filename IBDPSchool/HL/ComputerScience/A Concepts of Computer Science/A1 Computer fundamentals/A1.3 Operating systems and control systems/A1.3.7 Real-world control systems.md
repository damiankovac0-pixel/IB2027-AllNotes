# A1.3.7 Real-World Control Systems

Control systems are used wherever a computer monitors inputs and controls outputs in the physical world.

Use the structure from [[A1.3.6 Control system components|control system components]]: sensors, controller, algorithm, actuators, and feedback.

## Cruise Control

Cruise control maintains a vehicle's speed.

Input:

- driver sets target speed
- speed sensor measures current speed

Process:

- controller compares actual speed with target speed
- control algorithm calculates adjustment

Output:

- throttle actuator changes engine power

Feedback:

- speed is measured repeatedly, so the system adjusts if the car slows down or speeds up

This is a closed-loop system.

## Thermostat

A thermostat maintains a target temperature.

Input:

- temperature sensor measures room temperature
- user sets desired temperature

Process:

- controller compares measured temperature with target

Output:

- heater or air conditioner turns on/off or adjusts power

Feedback:

- temperature is measured again to check whether the target has been reached

## Irrigation System

An irrigation system controls watering.

Inputs:

- soil moisture sensor
- temperature sensor
- manual enable switch
- timer or weather data

Process:

- controller decides whether watering is needed

Output:

- valve or pump actuator starts/stops water flow

Feedback:

- soil moisture is measured again after watering

## Automatic Doors

Automatic doors open when a person or object is detected.

Inputs:

- motion sensor
- pressure mat
- proximity sensor

Process:

- controller decides whether the door should open or close

Output:

- motor opens or closes the door

Feedback:

- sensors check whether someone is still present before closing

## Autonomous Vehicles

Autonomous vehicles use complex control systems.

Inputs:

- cameras
- LiDAR
- radar
- GPS
- ultrasonic sensors
- inertial measurement units

Process:

- algorithms detect objects
- path planning calculates route
- localization estimates position
- decision-making systems choose speed, steering, and braking

Outputs:

- steering actuator
- throttle control
- braking system

Feedback:

- sensors constantly update the vehicle's understanding of the environment

This is a complex closed-loop system.

## Elevator Control

An elevator uses sensors and control logic to move safely between floors.

Inputs:

- floor buttons
- door sensors
- weight sensors
- position sensors

Process:

- controller decides direction, stops, and door operation

Output:

- motor moves elevator
- doors open or close

Feedback:

- position sensors confirm location and movement

## Exam Pattern

For any real-world control system, explain the components and the feedback loop. Strong answers show how sensor data leads to controller decisions and actuator actions.
