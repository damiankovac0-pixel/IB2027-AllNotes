# A1.4 Translation

Translation is the process of changing source code into instructions that a computer can execute.

This is an HL topic. The main focus is comparing translation methods, especially [[A1.4.1 Interpreters and compilers|interpreters and compilers]], and evaluating which method fits a scenario.

## Syllabus Parts

- [[A1.4.1 Interpreters and compilers|A1.4.1 Interpreters and compilers]]

## Big Picture

High-level programming languages are designed for humans to read and write. CPUs execute machine instructions encoded in binary. Translation bridges that gap.

```text
source code -> translation process -> machine-executable instructions
```
