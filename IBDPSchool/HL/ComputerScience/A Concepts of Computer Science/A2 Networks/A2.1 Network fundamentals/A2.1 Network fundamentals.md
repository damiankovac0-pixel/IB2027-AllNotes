# A2.1 Network Fundamentals

This topic introduces what networks are, why they are used, which devices make them work, and how protocols organize communication.

The big idea: a network connects devices so they can share data, resources, and services. This requires physical infrastructure, addressing, devices, protocols, and layered communication rules.

## Study Order

1. Learn network types: LAN, WAN, PAN, VPN, internet.
2. Learn infrastructure: internet, cloud, distributed systems, edge, mobile networks.
3. Learn devices: gateway, firewall, modem, router, switch, WAP.
4. Learn protocols: TCP, UDP, HTTP, HTTPS, DNS, DHCP.
5. Learn how the TCP/IP layers cooperate.
