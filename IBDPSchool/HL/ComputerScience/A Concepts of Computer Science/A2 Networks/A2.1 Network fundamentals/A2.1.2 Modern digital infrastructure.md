# A2.1.2 Modern Digital Infrastructure

Digital infrastructure is the physical and virtual foundation that allows digital data and services to operate.

It includes networks, servers, data centres, cloud platforms, mobile networks, edge devices, and distributed systems.

## Internet Infrastructure

The internet is a global infrastructure for communication and data exchange.

Benefits:

- global communication
- access to information and services
- collaboration across distance
- support for business, education, entertainment, and cloud services

Limitations:

- depends on reliable connectivity
- security risks
- privacy risks
- congestion and latency
- unequal access between regions

## Cloud Computing

Cloud computing provides computing resources over the internet.

This links to [[A1.1.9 Cloud services|A1.1.9 Cloud services]].

Benefits:

- scalable resources
- lower upfront hardware costs
- easier collaboration
- backup and disaster recovery
- managed maintenance by cloud providers

Limitations:

- requires reliable internet access
- ongoing costs
- possible latency
- data security and privacy concerns
- dependence on external providers
- possible regulatory issues if data is stored in another country

## Distributed Systems

A distributed system is a group of independent computers that work together and appear to users as one system.

The computers in the system are called nodes.

Benefits:

- more processing power through multiple nodes
- fault tolerance if one node fails
- scalability by adding more nodes
- can process data closer to where it is needed

Limitations:

- more complex to manage
- network failures can affect coordination
- data consistency can be difficult
- security must be handled across many nodes

## Edge Computing

Edge computing processes data close to where it is generated instead of sending everything to a central cloud server.

Examples:

- autonomous vehicles
- smart cameras
- IoT sensors
- health monitoring devices
- factory control systems

Benefits:

- lower latency
- less network traffic
- can improve privacy by keeping data local
- useful for real-time applications

Limitations:

- edge devices may have limited processing power
- many devices are harder to secure and maintain
- inconsistent hardware capabilities
- setup can be expensive

## Mobile Networks

Mobile networks provide wireless connectivity over large areas using cellular infrastructure.

Characteristics:

- supports movement between cells
- enables voice, data, and multimedia communication
- supports smartphones, tablets, and mobile IoT devices

Benefits:

- mobility
- wide coverage
- supports many users
- enables remote communication without wired connections

Limitations:

- signal quality varies
- congestion can reduce performance
- security risks from wireless transmission
- coverage gaps
- battery use on mobile devices

## Exam Pattern

For infrastructure questions, describe the purpose, benefits, and limitations. Strong answers compare performance, scalability, cost, latency, security, privacy, and reliability.
