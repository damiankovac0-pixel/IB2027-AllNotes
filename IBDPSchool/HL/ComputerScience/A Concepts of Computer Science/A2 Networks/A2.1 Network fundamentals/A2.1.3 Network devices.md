# A2.1.3 Network Devices

Network devices move, filter, convert, route, or connect data across networks.

In home networks, one physical box may act as modem, router, firewall, switch, and wireless access point. In exams, treat each device by its function.

## Gateway

A gateway connects networks that may use different protocols or formats.

Main role:

- bridge between networks
- translate or convert data where needed
- often connects a local network to another network or the internet

Example: a gateway can help an office network communicate with the internet and may perform NAT-related functions.

## Hardware Firewall

A hardware firewall monitors and controls incoming and outgoing traffic according to security rules.

Main role:

- block unauthorized or suspicious traffic
- allow permitted traffic
- create a barrier between trusted internal networks and untrusted external networks
- log traffic for monitoring

Rules may be based on:

- IP addresses
- ports
- protocols
- source and destination

Firewalls connect strongly to A2.4 Network security.

## Modem

Modem means modulator-demodulator.

A modem converts between digital computer signals and signals suitable for a communication medium.

Example: converting digital data into a form suitable for phone lines, cable, or other external service-provider infrastructure.

## Router

A router forwards packets between networks.

Main role:

- inspect packet destination information
- choose a route or next hop
- forward packets between networks
- connect LANs to wider networks

Example: if a computer sends data to a device outside its local network, the packet is sent to the default gateway/router.

## Switch

A switch connects devices within the same network segment.

Main role:

- forward frames to the specific device they are intended for
- reduce unnecessary traffic compared with broadcasting to all devices
- connect wired devices in a LAN

Switches can be used in homes, schools, offices, enterprise networks, and data centres.

## Wireless Access Point

A wireless access point, or WAP, connects wireless devices to a wired network.

Main role:

- broadcast Wi-Fi signal
- receive wireless data and pass it to wired network infrastructure
- convert between wireless radio communication and wired Ethernet communication
- extend network access to areas where cables are impractical

## Device Comparison

| Device | Main function |
| --- | --- |
| Gateway | connects or translates between networks |
| Firewall | filters traffic based on security rules |
| Modem | converts signal formats for transmission medium |
| Router | forwards packets between networks |
| Switch | connects devices inside a LAN segment |
| WAP | connects wireless devices to network |

## Packet Journey Example

If Host A sends data to Host B on another network:

1. Host A creates data for Host B.
2. Since Host B is outside the local network, Host A sends it to the router/default gateway.
3. Routers forward the packet across networks.
4. Each router uses destination information to choose the next hop.
5. The final network delivers the packet to Host B.

## Exam Pattern

For device questions, identify what problem the device solves. Do not rely on home-router names, because one physical device can contain several logical network device functions.
