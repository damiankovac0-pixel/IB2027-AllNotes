# A2.2 Network Architecture

Network architecture is the overall design of a network: its devices, topology, servers, models, segmentation, and how components interact.

The big idea: good network architecture balances performance, reliability, scalability, cost, security, and manageability.

## Key Distinction

Physical topology describes the actual layout of cables, devices, and connections.

Logical topology describes how data flows through the network, regardless of the physical layout.
