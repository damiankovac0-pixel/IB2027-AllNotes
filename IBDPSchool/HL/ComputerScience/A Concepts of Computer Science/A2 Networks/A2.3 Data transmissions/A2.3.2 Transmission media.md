# A2.3.2 Transmission Media

Transmission media are the physical or wireless channels used to carry data between devices.

The main media in this syllabus are:

- fibre-optic cable
- twisted-pair cable
- wireless transmission

## Comparison Factors

When comparing media, consider:

- bandwidth
- latency
- cost
- installation complexity
- range and attenuation
- susceptibility to interference
- security
- reliability
- suitability for the scenario

## Fibre-Optic Cable

Fibre-optic cable transmits data as pulses of light through glass or plastic fibres.

Advantages:

- very high bandwidth
- very long range
- low attenuation over distance
- not affected by electromagnetic interference
- difficult to tap without detection
- suitable for high-speed backbone links and long-distance connections

Disadvantages:

- higher cost than twisted pair
- more complex installation
- specialist tools and skills may be needed
- cable can be fragile if bent incorrectly

Typical uses:

- internet backbone
- submarine cables
- data centres
- connections between buildings
- high-speed enterprise networks

## Twisted-Pair Cable

Twisted-pair cable uses pairs of insulated copper wires twisted together.

Types:

- UTP: unshielded twisted pair
- STP: shielded twisted pair

Advantages:

- relatively cheap
- widely available
- easy to install
- flexible
- good enough for many homes and offices
- supports high speeds over short distances

Disadvantages:

- lower bandwidth than fibre
- shorter range, often around 100 metres for Ethernet runs
- affected by electromagnetic interference, especially UTP
- easier to tap than fibre

Typical uses:

- office LANs
- home Ethernet
- school networks
- connecting PCs, switches, printers, and access points

## Wireless Transmission

Wireless transmission uses electromagnetic waves, such as radio waves, to send data without cables.

Examples:

- Wi-Fi
- Bluetooth
- mobile networks
- satellite links

Advantages:

- mobility
- no cable installation to each device
- convenient for phones, tablets, laptops, and IoT devices
- useful where cabling is impractical

Disadvantages:

- more vulnerable to interference
- signal weakens with distance and obstacles
- security risks because signals can be intercepted without physical access
- bandwidth and latency vary with conditions
- shared medium can become congested

Typical uses:

- home Wi-Fi
- campus networks
- mobile devices
- public hotspots
- Bluetooth PANs

## Interference

Interference can reduce signal quality.

Fibre-optic cable is not affected by electromagnetic interference because it uses light.

Twisted-pair cable can be affected by nearby electrical equipment, especially if unshielded.

Wireless can be affected by walls, distance, other wireless devices, and sources operating on similar frequencies.

## Attenuation

Attenuation is signal loss over distance.

- fibre has low attenuation
- twisted pair has moderate attenuation and limited range
- wireless weakens with distance and obstacles

## Security

Fibre is generally harder to intercept physically.

Twisted pair requires physical access to cable or network equipment.

Wireless can be intercepted from nearby unless encryption and authentication are strong.

## Summary Table

| Medium | Strengths | Weaknesses | Best use |
| --- | --- | --- | --- |
| Fibre-optic | fastest, long range, EMI-resistant | costly, complex installation | backbone, data centres, long distance |
| Twisted-pair | cheap, common, easy to install | shorter range, EMI risk | LANs, homes, offices |
| Wireless | mobile and convenient | interference/security/range issues | mobile devices, Wi-Fi, hard-to-cable areas |

## Exam Pattern

For media comparison questions, avoid saying one medium is always best. Explain which factors matter for the scenario: speed, cost, range, installation, interference, security, and mobility.
