# A2.4 Network Security

Network security protects data, devices, services, and users from unauthorized access, disruption, interception, and damage.

The key idea is defence in depth: no single tool protects a network completely. Firewalls, authentication, encryption, monitoring, patching, training, and backups work together.

## Study Order

1. Understand what firewalls can and cannot do.
2. Learn common vulnerabilities and attacks.
3. Match each vulnerability to suitable countermeasures.
4. Understand encryption, public/private keys, certificates, and digital signatures.

## Exam Focus

Network security questions often ask for evaluation, not just definitions.

Strong answers usually mention:

- the specific threat
- the specific countermeasure
- why it reduces risk
- what it does not protect against
- whether it suits the scenario
