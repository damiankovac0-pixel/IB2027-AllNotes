# A3.3 Database Programming

Database programming uses SQL to define database structures, retrieve data, update data, calculate summaries, create views, and manage reliable transactions.

## Study Order

1. Learn SQL language categories: DDL, DML, DCL, and TCL.
2. Practise SELECT queries with WHERE, JOIN, ORDER BY, GROUP BY, LIKE, AND, OR, and DISTINCT.
3. Learn INSERT, UPDATE, and DELETE.
4. Practise aggregate functions with GROUP BY.
5. Understand views as virtual tables.
6. Understand ACID transactions.
