# A4.1 Machine Learning Fundamentals

This topic introduces what [[A4.1.0 AI and ML foundations#Machine Learning|machine learning]] is, how it fits inside [[A4.1.0 AI and ML foundations#Artificial Intelligence|artificial intelligence]], the main ways models learn, common real-world uses, and the hardware needed to run them.

The core idea: a [[A4.1.0 AI and ML foundations#Machine Learning Model|machine learning model]] learns patterns from data instead of relying only on rules manually written by a programmer.

## Big Picture

[[A4.1.0 AI and ML foundations#Artificial Intelligence|AI]] is the broad field. [[A4.1.0 AI and ML foundations#Machine Learning|Machine learning]] is a branch of AI where systems learn from data. [[A4.1.1 Types of machine learning#Deep Learning|Deep learning]] is a branch of machine learning that uses multi-layer [[A4.1.1 Types of machine learning#Neural Network|neural networks]].

```text
Artificial intelligence
+-- Machine learning
    +-- Deep learning
```

The five learning types used in this topic are:

- [[A4.1.1 Types of machine learning#Supervised Learning|supervised learning]]
- [[A4.1.1 Types of machine learning#Unsupervised Learning|unsupervised learning]]
- [[A4.1.1 Types of machine learning#Reinforcement Learning|reinforcement learning]]
- [[A4.1.1 Types of machine learning#Deep Learning|deep learning]]
- [[A4.1.1 Types of machine learning#Transfer Learning|transfer learning]]
