# A4.1.0 AI And ML Foundations

This note gives the basic vocabulary needed before comparing the types of machine learning.

## Artificial Intelligence

Artificial intelligence is the broad field of computer systems that can perform tasks that normally seem to require human intelligence.

Examples of AI-like tasks:

- recognizing images
- understanding speech or text
- making decisions
- solving problems
- generating text, images, audio, or video

Two common forms are:

- **Narrow AI**: designed for a specific task, such as classifying images or recommending videos. This is what current systems actually do.
- **General AI**: human-level intelligence across many domains. This is still speculative.

## Machine Learning

Machine learning is a branch of [[#Artificial Intelligence|artificial intelligence]] where computers learn patterns from data and experience instead of being explicitly programmed with every rule.

Traditional programming:

```text
rules + data -> answer
```

Machine learning:

```text
data + training process -> model -> predictions on new data
```

Machine learning is commonly used to:

- predict values
- classify items
- find patterns
- recommend products or media
- detect objects, fraud, medical conditions, or unusual behaviour

## Machine Learning Algorithm

A machine learning algorithm is the method used to learn from data.

The algorithm is the learning procedure. After it has learned from data, the trained result is the [[#Machine Learning Model|model]].

## Machine Learning Model

A machine learning model is the trained result of a machine learning algorithm.

Example: an algorithm trains on many labelled emails. After training, the model can classify a new email as spam or not spam.

## Data Set

A data set is a collection of data used for training, testing, or evaluating a model.

Examples:

- emails
- medical scans
- house sales
- customer transactions
- student grades and attendance
- social media comments

The quality of the data set strongly affects the model. This is why [[A4.2 Data preprocessing|data preprocessing]] matters.

## Labelled Data

Labelled data contains examples where the correct answer is already known.

Examples:

- an image labelled `cat`
- an email labelled `spam`
- a scan labelled with a diagnosis
- a house record labelled with the sale price

[[A4.1.1 Types of machine learning#Supervised Learning|Supervised learning]] uses labelled data. [[A4.1.1 Types of machine learning#Unsupervised Learning|Unsupervised learning]] does not.

## Training

Training is the process where an algorithm learns patterns from data.

During training, the model adjusts internally so its outputs become more useful or accurate. After training, the model is tested or used on new data.

## Generative AI

Generative AI is AI that creates new content, such as text, images, audio, video, or code, usually in response to a prompt.

For exams, the important point is not memorizing current tools. It is understanding that many generative AI systems are built using machine learning, especially [[A4.1.1 Types of machine learning#Deep Learning|deep learning]].
