# A4.1.1 Real-World ML Applications

This note connects the learning types from [[A4.1.1 Types of machine learning]] to real examples.

## Market Basket Analysis

Market basket analysis finds products that are often bought together.

Example: customers who buy pasta may also buy tomato sauce.

Best fit: [[A4.1.1 Types of machine learning#Unsupervised Learning|unsupervised learning]].

Why:

- there is no correct label to predict
- the goal is to discover hidden associations in transaction data
- algorithms such as Apriori or FP-Growth can find frequent itemsets

Exam phrasing: it discovers association rules such as `X -> Y`, meaning customers who buy X are likely to buy Y.

Ethical point: loyalty cards and online accounts help businesses collect detailed behavioural data, which creates privacy concerns.

## Medical Imaging Diagnostics

Medical imaging diagnostics uses models to analyse scans such as X-rays, MRIs, and CT scans.

Best fit: [[A4.1.1 Types of machine learning#Supervised Learning|supervised learning]], [[A4.1.1 Types of machine learning#Deep Learning|deep learning]], and [[A4.1.1 Types of machine learning#Transfer Learning|transfer learning]].

Why supervised learning:

- scans can be labelled with diagnoses by medical experts
- the task is often classification, such as disease present or not present

Why deep learning:

- scans contain complex visual patterns
- CNN-style models can learn image features automatically

Why transfer learning:

- labelled medical data can be limited
- a model pre-trained on general image data can be fine-tuned for a medical task

## Natural Language Processing

Natural language processing, or NLP, is the use of computers to process and understand human language.

Examples:

- translation
- chatbots
- text summarization
- grammar feedback
- question answering

Best fit: [[A4.1.1 Types of machine learning#Deep Learning|deep learning]], [[A4.1.1 Types of machine learning#Supervised Learning|supervised learning]], and [[A4.1.1 Types of machine learning#Transfer Learning|transfer learning]].

Why:

- language depends on sequence and context
- labelled text can be used for tasks such as translation or classification
- pre-trained language models can be fine-tuned for specific tasks

Transformers are especially important because they use attention to weigh the importance of words in context.

## Object Detection and Classification

Object classification identifies what object is in an image. Object detection also identifies where it is, often using bounding boxes.

Best fit: [[A4.1.1 Types of machine learning#Supervised Learning|supervised learning]] and [[A4.1.1 Types of machine learning#Deep Learning|deep learning]].

Why:

- training images can be labelled with object categories and locations
- deep learning models can learn visual hierarchies from simple edges to complete objects

Examples:

- facial recognition
- traffic sign detection
- identifying objects in security footage
- detecting pedestrians for autonomous vehicles

## Robotics Navigation

Robotics navigation is about robots learning how to move through and respond to their environment.

Best fit: [[A4.1.1 Types of machine learning#Reinforcement Learning|reinforcement learning]] and [[A4.1.1 Types of machine learning#Supervised Learning|supervised learning]].

Why reinforcement learning:

- robots make sequences of decisions
- they can improve through rewards and penalties
- trial and error helps with path planning and obstacle avoidance

Why supervised learning:

- sensor or camera data can be labelled for tasks such as obstacle recognition
- the robot can classify immediate surroundings quickly

## Sentiment Analysis

Sentiment analysis classifies the emotional tone of text, usually as positive, negative, or neutral.

Best fit: [[A4.1.1 Types of machine learning#Supervised Learning|supervised learning]], [[A4.1.1 Types of machine learning#Deep Learning|deep learning]], and [[A4.1.1 Types of machine learning#Transfer Learning|transfer learning]].

Why:

- reviews or comments can be labelled by sentiment
- deep learning can handle word order, context, and nuance
- transfer learning lets a general language model adapt to a specific domain, such as movie reviews or social media posts
