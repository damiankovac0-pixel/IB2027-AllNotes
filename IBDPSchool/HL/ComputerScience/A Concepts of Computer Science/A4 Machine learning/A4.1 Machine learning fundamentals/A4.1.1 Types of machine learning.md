# A4.1.1 Types Of Machine Learning

This syllabus point is about knowing the main learning types and when each one is suitable.

## Supervised Learning

Supervised learning trains on [[A4.1.0 AI and ML foundations#Labelled Data|labelled data]]. The model is shown inputs together with the correct outputs, then learns to predict outputs for new data.

Use it when the desired answer is known during training.

Typical tasks:

- [[#Regression|regression]]
- [[#Classification|classification]]

Examples:

- predicting a house price from property data
- classifying an email as spam or not spam
- diagnosing a medical scan from labelled examples

## Regression

Regression is a supervised learning task where the output is a number.

Examples:

- house price
- predicted temperature
- final exam score
- customer spending

The key test: if the answer is a continuous numerical value, it is probably regression.

## Classification

Classification is a supervised learning task where the output is a category.

Examples:

- spam or not spam
- high risk, medium risk, or low risk
- cat, dog, or muffin
- positive, negative, or neutral

The key test: if the answer is a label from a set of categories, it is classification.

## Unsupervised Learning

Unsupervised learning uses data without labelled answers. The model tries to find structure by itself.

Use it when the goal is to discover patterns rather than predict a known label.

Examples:

- grouping customers with similar buying habits
- discovering products often bought together
- finding social groups from user interactions
- clustering data before analysis

Common unsupervised tasks include clustering, association rules, and dimensionality reduction.

## Reinforcement Learning

Reinforcement learning trains an agent through trial and error. The agent takes actions in an environment and receives rewards or penalties.

Important terms:

- **agent**: the learner or decision-maker
- **environment**: the world the agent acts in
- **action**: what the agent chooses
- **reward**: feedback on whether the action was good or bad
- **policy**: the strategy the agent learns

Use it when a system needs to learn sequences of decisions.

Examples:

- game-playing AI
- robot movement
- autonomous vehicle navigation
- adaptive learning pathways
- trading bots receiving profit/loss feedback

## Deep Learning

Deep learning is a type of [[A4.1.0 AI and ML foundations#Machine Learning|machine learning]] that uses multi-layer [[#Neural Network|neural networks]].

It is useful for complex data where manual rule-writing is unrealistic, such as images, speech, and language.

Examples:

- image recognition
- speech recognition
- natural language processing
- medical image analysis
- object detection

Do not treat deep learning as separate from machine learning. It is a subset of machine learning.

## Neural Network

A neural network is a model inspired by the idea of connected neurons.

It contains layers of nodes that process inputs and pass outputs onward. Earlier layers may detect simple features; later layers can combine these into more complex patterns.

For example, in image processing:

- early layers may detect edges
- middle layers may detect shapes
- later layers may detect objects

## Transfer Learning

Transfer learning uses a model trained for one task as the starting point for a related task.

The point is to reduce the amount of data, time, and computing power needed for the new task.

Examples:

- adapting a general image model to identify flower species
- fine-tuning a language model for customer support
- adapting a driving model trained in one city to another city
- adapting a speech model to a specific accent

Exam warning: transfer learning is not just using a pre-trained model unchanged. It involves adapting or fine-tuning it for a related task.
