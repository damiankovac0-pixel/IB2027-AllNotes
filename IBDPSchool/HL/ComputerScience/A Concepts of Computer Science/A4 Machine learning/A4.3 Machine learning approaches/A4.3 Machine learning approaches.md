# A4.3 Machine Learning Approaches

This topic moves from the general types of [[A4.1.1 Types of machine learning|machine learning]] to specific approaches and algorithms.

The main skill is matching the problem type to a suitable approach, then explaining how the approach works and how its performance is evaluated.
