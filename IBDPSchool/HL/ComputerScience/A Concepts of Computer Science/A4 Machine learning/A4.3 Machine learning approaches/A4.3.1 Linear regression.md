# A4.3.1 Linear Regression

Linear regression is a [[A4.1.1 Types of machine learning#Regression|regression]] method used to predict a continuous numerical outcome.

Examples:

- predicting house price from floor area
- predicting customer spending from past purchases
- predicting temperature from weather data
- predicting exam score from attendance and study time

## Core Idea

Linear regression assumes there is a linear relationship between the input variable or variables and the output.

For one input variable, the idea is:

```text
Y = b0 + b1X + error
```

Where:

- `Y` is the dependent variable, the value being predicted
- `X` is the independent variable, the input used to predict Y
- `b0` is the intercept
- `b1` is the slope
- `error` is the part not explained by the line

## Dependent And Independent Variables

The **dependent variable** is what the model is trying to predict.

The **independent variable** is the input used to explain or predict the dependent variable.

Example: predicting house price from floor area.

- dependent variable: house price
- independent variable: floor area

## Slope And Intercept

The **intercept** is the predicted value of `Y` when `X` is 0.

The **slope** shows how much `Y` changes when `X` increases by one unit.

Example: if slope is `1600`, then each extra square metre of floor area increases the predicted house price by about `1600`, assuming the model is appropriate.

## Assessing The Fit

A regression line should fit the data well enough to make useful predictions. If the data points are very far from the line, the model is weak.

Common measures include:

- residuals: differences between actual and predicted values
- mean absolute error
- mean squared error
- R-squared

## R-Squared

R-squared measures how much of the variation in the dependent variable is explained by the model.

- `R2 = 0` means the model explains none of the variation
- `R2 = 1` means the model explains all of the variation

High R-squared can suggest a better fit, but it does not automatically prove the model is good. A model can still be biased, overfitted, or based on poor features.

## Problems To Watch For

### Outliers

Outliers can pull the regression line away from the main pattern and reduce prediction quality.

This links to [[A4.2.1 Data cleaning#Outliers|data cleaning]].

### Multicollinearity

Multicollinearity happens when independent variables are highly correlated with each other.

Example: house floor area and number of rooms may both describe house size. If both are used, the model may struggle to separate their effects.

Ways to reduce it:

- remove one of the correlated features
- combine features
- use [[A4.2.2 Feature selection|feature selection]]
- use [[A4.2.3 Dimensionality reduction|dimensionality reduction]]

## Exam Pattern

Linear regression predicts continuous numerical outcomes by fitting a line that models the relationship between independent variables and a dependent variable. The slope shows the change in the predicted output for each unit change in the input, and the intercept is the predicted output when the input is zero.
