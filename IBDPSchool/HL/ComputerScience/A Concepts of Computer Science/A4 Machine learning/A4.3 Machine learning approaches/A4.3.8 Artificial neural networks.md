# A4.3.8 Artificial Neural Networks

An artificial neural network, or ANN, is a model made of connected nodes arranged in layers.

This expands the basic idea of [[A4.1.1 Types of machine learning#Neural Network|neural networks]].

## Structure

Typical layers:

- **input layer**: receives the input features
- **hidden layers**: process patterns in the data
- **output layer**: produces the prediction or classification

More hidden layers allow the network to model more complex patterns, but they also increase training cost and overfitting risk.

## Node Or Perceptron

A perceptron is a basic node in a neural network.

It takes inputs, multiplies them by weights, adds a bias, applies an activation function, and outputs a value.

```text
inputs -> weights -> weighted sum + bias -> activation function -> output
```

## Weights

Weights control the importance of inputs. During training, the network adjusts weights to reduce error.

## Bias

Bias is an added value that shifts the output of a node. It helps the model fit patterns that do not pass through zero.

## Activation Function

An activation function decides how the node's calculated value is transformed before passing onward.

Activation functions introduce non-linearity, allowing neural networks to learn complex patterns.

Examples:

- ReLU
- sigmoid
- tanh
- linear activation for some regression outputs

## Forward Propagation

Forward propagation is when data moves from the input layer through hidden layers to the output layer.

The model produces a prediction.

## Loss

Loss measures how far the prediction is from the correct answer.

Training tries to reduce this loss.

## Backpropagation

Backpropagation is the learning process where the network adjusts weights and biases to reduce error.

Basic idea:

1. make a prediction using forward propagation
2. calculate the error or loss
3. work backward through the network
4. adjust weights and biases to reduce future error

## Single-Layer vs Multi-Layer Networks

A single perceptron can only separate linearly separable data.

A multi-layer network can learn non-linear patterns because hidden layers and activation functions allow more complex decision boundaries.

## Uses

ANNs can be used for:

- classification
- regression
- pattern recognition
- risk prediction
- image, text, and signal processing

## Challenges

- can require large data sets
- can require significant hardware, linking to [[A4.1.2 Hardware for machine learning|ML hardware]]
- less interpretable than simple models
- can overfit
- training can be slow

## Exam Pattern

An ANN consists of input, hidden, and output layers. Nodes use weights, biases, and activation functions to transform inputs. Forward propagation produces an output, and backpropagation adjusts weights and biases to reduce prediction error.
