# A4.3.9 Convolutional Neural Networks

A convolutional neural network, or CNN, is a neural network designed especially for image data.

CNNs learn spatial hierarchies of features. This means they can learn simple image features first, then combine them into more complex features.

Example:

```text
edges -> shapes -> object parts -> full object
```

## Why CNNs Are Useful For Images

Images contain spatial relationships. Nearby pixels often matter together.

A normal neural network treats each input more separately, but a CNN uses filters to scan small regions of the image and detect local patterns.

## Input Layer

The input layer receives raw pixel data.

For a colour image, input includes width, height, and colour channels.

## Convolutional Layer

A convolutional layer applies filters, also called kernels, across the image.

Each filter detects a feature, such as an edge, corner, texture, or shape.

The result is a feature map showing where that feature appears.

## Activation Function

Activation functions such as ReLU add non-linearity, helping the CNN learn more complex patterns.

## Pooling Layer

Pooling reduces the spatial size of feature maps.

This reduces computation and can make the model more robust to small shifts in the image.

Common example: max pooling keeps the strongest value in a region.

## Flattening

Flattening converts the processed feature maps into a one-dimensional form so they can be passed to dense layers.

## Dense Layers

Dense layers use the learned image features to make the final classification or prediction.

## Softmax Output

For multi-class classification, softmax can convert output scores into probabilities that sum to 1.

Example: cat `0.80`, dog `0.15`, rabbit `0.05`.

## Factors Affecting CNN Performance

### Number Of Layers

Fewer layers may only learn simple features. More layers can learn complex features, but require more data and computing power.

### Filter Size And Number

More filters can detect more features, but increase computation.

### Training Data Quality

CNNs need relevant and representative data. Poor images, biased labels, or missing examples reduce performance.

This links to [[A4.2 Data preprocessing|data preprocessing]].

### Hardware

CNN training often benefits from [[A4.1.2 Hardware for machine learning#GPU|GPUs]] because convolution operations are computationally heavy.

## Uses

CNNs are used in:

- medical image analysis
- object detection
- facial recognition
- autonomous vehicle vision
- quality inspection in manufacturing
- image classification

## Exam Pattern

CNNs adaptively learn spatial hierarchies by applying filters in convolutional layers to detect local image features, reducing feature maps with pooling, and using later layers to classify based on increasingly complex patterns.
