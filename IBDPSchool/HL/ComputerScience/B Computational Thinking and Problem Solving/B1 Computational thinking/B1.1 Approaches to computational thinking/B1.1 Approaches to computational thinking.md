# B1.1 Approaches To Computational Thinking

Computational thinking is a framework for solving problems by applying ideas from computer science. It is not about writing code — it is about how to think about problems so that solutions can be represented and executed computationally.

## Study Order

1. Learn how to construct a proper problem specification.
2. Master the four core concepts: decomposition, pattern recognition, abstraction, algorithmic design.
3. See how these concepts apply in software development, data analysis, network security, and game development.
4. Practise reading and tracing flowcharts with variables, decisions, and loops.
