One Obsidian vault with everything needed to score **6–7** on IB DP finals.

> If you know all of it truly.

Subjects currently included:
- **Computer Science HL** — A1 (Computer fundamentals), A2 (Networks), A3 (Databases), A4 (Machine learning) complete with hub + subtopic notes. B section (Computational thinking, Programming, OOP, ADTs) will be filled in order.

## Goal

Every topic, every syllabus point, every exam pattern — explained clearly, cross-linked, and split into focused notes that are actually readable. No textbooks to flip through, no scattered PDFs, no cramming from half-finished summaries.

One vault. Know it all. Walk into the exam ready.

## Structure

- Topics split into **hub notes** (syllabus overview + links) and **subtopic notes** (deep dives per syllabus point).
- Key concepts linked across topics via `[[Obsidian wikilinks]]`.
- Graphs grouped by colour in Obsidian for easy navigation.
